# Analisi Dati Umanitari

Repository per l'analisi esplorativa di dataset umanitari (UNICEF e popolazione).

## Dataset presenti

- `fusion_GLOBAL_DATAFLOW_UNICEF_1.0_.CME_MRY1T4..csv`
- `fusion_GLOBAL_DATAFLOW_UNICEF_1.0_.CME_MRY5T14..csv`

Costruire analisi descrittive e visualizzazioni utili per comprendere trend su mortalita' infantile e contesto demografico.
